## Design

## Architecture

- **App.jsx**: Main state management and logic
- **TodoInput**: Input form component for adding tasks
- **TodoList**: Container for todo items
- **TodoItem**: Individual task component with toggle and delete

All data is persisted to browser localStorage automatically.
